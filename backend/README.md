# AI SaaS Backend

Production-ready backend for a SaaS dashboard application.

## Features

- JWT Authentication
- Protected Routes
- CRUD API
- Prisma ORM
- PostgreSQL
- Rate Limiting
- Validation with Zod
- Error Handling
- Pagination

## Tech Stack

- Node.js
- Express
- Prisma
- PostgreSQL
- TypeScript

## API Structure

/routes
/controllers
/services
/middlewares
/utils

## Setup

npm install
npm run dev
