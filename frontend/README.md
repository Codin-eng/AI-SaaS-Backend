# AI SaaS Frontend

Modern SaaS dashboard frontend connected to a production backend API.

## Features

- Next.js 16
- TypeScript
- Tailwind CSS
- JWT Authentication
- Protected Routes
- Dashboard UI
- CRUD Integration
- Axios Interceptors
- Loading States
- Toast Notifications

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Axios

## Setup

npm install
npm run dev
